## NYU Libraries Stylesheets

Global stylesheets for NYU Libraries suite of search applications, Bobcat. Fluid and responsive Web design. Written in SCSS<sup>1</sup> using the Compass framework<sup>2</sup> and the Blueprint CSS bootstrap<sup>3,4,5</sup>.

1. http://sass-lang.com/
2. http://compass-style.org/
3. http://compass-style.org/reference/blueprint/
4. http://compass-style.org/reference/blueprint/liquid/
5. http://blueprintcss.org/
